# Fundamentos de Análise de Dados

Este repositório reúne estudos e exercícios práticos voltados ao desenvolvimento das habilidades essenciais em análise de dados.

O foco aqui é consolidar a base técnica necessária antes da construção de projetos mais completos e orientados a problemas reais.

## 📂 Estrutura do Repositório

### Extração de Dados

* `extracao_catalogo_livros.py`
  Script para coleta de dados de um site utilizando requisições a uma API.

* `ecommerce_sql/`
  Script com consultas SQL para extração de informações em banco de dados relacional

### Visualização de Dados

* `graficos/`
  Exercícios de criação de gráficos utilizando **Seaborn** e **Matplotlib (pyplot)**.

### Tratamento de Dados

* `tratamento_dados/` e `ecommerce_dados/`
Exercícios focados em limpeza de dados e preparação de datasets para análise

### Análise Exploratória Inicial

* `comparacao_categorias_mercado/`

Análise simples comparando média e mediana de preços por categoria, com visualizações e interpretação inicial dos dados.
